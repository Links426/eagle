# 什么是 Eagle 🦅
Cats. 全程 Enable Agile Generation and Linking of Entries； 读起来类似于英文单词 Eagle （鹰）, 是一款 支持一键更新远端种子项目的脚手架CLI。

Cats 主要包括了两部分：
1. 根据 种子项目 新建项目。
2. 在新建的项目，使用命令，一键拉取种子项目的更新。

### 主要特性
1. 使用命令，一键式的创建项目
2. 支持和远端项目列表的同步和更新
3. 一键更新拉取种子项目的更新
4. 文件级别的更新可选项
5. 毫无心智负担的易用性
